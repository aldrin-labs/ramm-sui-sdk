# `ramm-sui-sdk`

## Building the library

`yarn`

## Testing the library

`yarn test`

Note that tests require a `RAMM_SUI_SDK_PRIVATE_KEY` variable present in the shell's environment.
The key must have a sufficient `Coin<SUI>` balance for trades - in the testnet.
