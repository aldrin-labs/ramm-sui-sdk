# `ramm-sui-sdk`

## Building the library

`yarn`

## Testing the library

`yarn test`

Note that tests require a `.env` file in the root of the repository with a Sui `PRIVATE_KEY` (testnet),
with a sufficient `Coin<SUI>` balance for trades.
